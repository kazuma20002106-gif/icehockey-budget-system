# Cycle 12 年度末出力の分割提案

年度末決算ファイル一括出力は、DB・Excelセル座標・金額計算・UIを同時に変更する大きな危険タスクです。

一度に全部実装すると、どこか1セルでもズレたときに原因を追いにくくなります。

Dexとしては以下の3分割を提案します。

1. **12A: Excel土台**
   - 実テンプレートのシート名・セル座標を確定
   - 2-1/2-2/2-2-1/2-4/2-5/2-6の出力順と既存数式保持を確認

2. **12B: 予算管理と2-3**
   - `budget_allocations` テーブル
   - 予算管理画面
   - 2-3の内示額・変更実績出力

3. **12C: プレビュー改善**
   - 大容量プレビューのタブ化
   - 出力パラメータダイアログ

この順番なら、金額とセル座標の安全確認を先に固めてから、UIを広げられます。
