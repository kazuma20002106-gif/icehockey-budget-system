# CC Cycle 5 提案

## 提案1: 様式2-6の雑費合計について

現在の実装: 雑費欄は画面表示「-」のため、合計セル(row39, col27)には0を書き込んでいる。旅費合計にはDBの`miscellaneousCost`を含めている。

**提案**: 将来的に雑費が正式に使われる場合、雑費欄の「-」表示を廃止し数値表示にすることを検討する。その際、合計セルも自動更新される。

## 提案2: 宿泊日フィールドの追加

現状、宿泊日は`eventDate`から推測（eventDate当日）しているが、実際には翌日まで泊まる場合などがある。

**提案**: `projects`テーブルに`stay_nights`（泊数）または`end_date`カラムを追加し、様式2-5の宿泊日ヘッダーを正確に表示する。

## 提案3: 活動一覧の一括プレビュー出力

現在は一覧から単一活動のプレビューのみ。複数チェックして一括プレビューする機能があると便利。

**提案**: 一覧にチェックボックスを追加し、選択した複数活動を一括でプレビュー→ダウンロードできる導線を追加する（既存の`/export`画面の機能を一覧から使えるようにする）。
