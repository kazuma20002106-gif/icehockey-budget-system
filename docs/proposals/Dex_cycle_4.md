# Dex ＋α提案 — サイクル4

## 提案

操作ユーザー設定は今後の帳票自動印字の土台になるため、次サイクル以降で「帳票プレビュー画面に、今回印字される記入責任者氏名・電話番号を事前表示する」改善を検討するとよいです。

これにより、Excel出力後に開いて確認する前の段階で、操作者の切り替え漏れに気づきやすくなります。

## P4レビュー時の追加提案

共通レイアウトに新しいモデル属性を追加した場合は、今後のサイクルで簡単な画面表示テスト（`/` と `/activity` のテンプレート描画確認）を1本追加すると、コンパイルでは拾えないThymeleaf式の不具合を早期に検出できます。
