# Dex Cycle 8.3 Take1 レビュー後の提案

今回、`buildCombinedWorkbook` と `exportForm24/exportForm25/exportForm26` のように、似たExcel出力経路が複数あり、一方だけ修正されるリスクが見えました。

次回以降のExcel改修では、P1/P2に「対象となる全エクスポート経路」を明示するチェック項目を追加することを提案します。

例:

- 年度まとめExcel
- 単一活動のまとめ出力
- 複数選択のall出力
- 複数選択の2-4のみ
- 複数選択の2-5のみ
- 複数選択の2-6のみ

これを最初から列挙しておくと、「片方の導線だけ直っている」事故を減らせます。
