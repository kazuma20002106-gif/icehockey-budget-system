# Cycle 4: ユーザー設定・エクセル自動印字・UI修正（P1 -> P2）

Dexへ。以下の4つの要件を満たすP2指示書（`docs/handoff/P2_Dex_to_CC/cycle_4.md`）を作成してください。

## ① 過去データのDBマイグレーション
- CCへ、既存の `expenses` テーブルの `transport_method` カラムの値を更新するSQL（マイグレーション処理）を実行（またはスクリプト作成）するよう指示すること。
  - `電車・車` → `電車`
  - `航空機・バス` → `航空機`
- これはCycle 3の変更に伴うデータ整合性確保のため。

## ② 様式2-6（および2-5）の氏名欄の幅修正
- `form.html` において、氏名を入力・表示する列（`.name-mirror` などの input、およびその親の `th`, `td`）に対し、CSS（インラインスタイルまたは `style.css`）で `min-width: 140px;` などを指定し、長い名前（例：カズマック）が見切れないように修正すること。

## ③ 簡易ログイン（利用者設定）とエクセル自動印字
- **DB変更** (`schema.sql` 等):
  - `users` テーブル（`id`, `name`, `phone_number` 等）を作成。
  - `system_settings` テーブル（`setting_key`, `setting_value`）を作成し、現在のログインユーザーIDを保持する。
  - 初期データとして `users` に「齋藤 和明（090-5288-9928）」を登録し、`system_settings` のアクティブユーザーに設定する。
- **UI変更** (`layout.html` 等):
  - 画面上部のナビゲーションバーなどに「現在の操作者：〇〇 ▼」のようなプルダウン（またはボタン＋Modal）を配置し、パスワードなしでアクティブユーザーを切り替え・新規登録・編集できるようにする。
- **Excel連携** (`ExcelExportService.java` 等):
  - 様式2-4の最下段「記入責任者氏名」「電話番号」のセルに、現在アクティブに設定されているユーザーの氏名と電話番号を自動印字する。

## ④ 様式2-4の「〇つけ機能」の座標微調整
- `ExcelExportService.java` の `drawEllipse` メソッドが、様式2-4の最新エクセルテンプレート上で正しい文字（強化練習、成年男子 等）を正確に丸で囲めているか、座標（row/col）を検証・微調整すること。
